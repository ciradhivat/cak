# รวมคำศัพท์ภาษาผัก (Bhasa Phak Glossary)

**หมายเหตุ:** ศัพท์ใดจำเป็นต้องใช้แต่ไม่มี ให้ยืมภาษาไทยมาใช้ได้ เช่น น้ำเงิน N̋amɲean

## สัญลักษณ์อักขรวิธี
- ṅ, ง → ɲ
- ñ, ญ → ɳ
- ศ → z̥
- ษ → z
- ฮ → xh
- บ → p̥
- ฎ → d̥
- ฟ → b̥
- ฝ → p̥h
- ฤ → t̥h
- ฦ → b̥h

**วรรณยุกต์:**
- k̍/k̀/ḱ k̎/k̏/k̋ ǩ k̽
- ก่/ก่/ก่ ก้/ก้/ก้ ก๊ ก๋

---

## 1. หมวดสรรพนาม (Pronouns)
- **Aha** (อะหะ) = ฉัน
- **Tava** (ตะวะ) = เธอ
- **So** (โส) = เขา
- **Ya** (ยะ) = ผู้ / ที่ / ซึ่ง / อัน (สรรพนามเชื่อมความ)
- **Sabba** (สัพพะ) = ทั่วไป / ทั้งปวง
- **Ghaṇa** (คะณะ) = พวก
- **Eta** (เอตะ) = นั้น
- **Amo** (อะโม) = โน้น

---

## 2. หมวดคำกริยา (Verbs)
- **Harati** (หะระติ) = นำ / พา
- **Karati** (กะระติ) = ทำ
- **Gacchati** (คัจฉะติ) = ไป
- **Passati** (ปัสสะติ) = เห็น
- **Pivati** (ปิวะติ) = ดื่ม
- **Vadati** (วะทะติ) = พูด
- **Suṇati** (สุณะติ) = ฟัง
- **Hoti** (โหติ) = เป็น / อยู่ / คือ / มี
- **Bhuñjati** (ภุญชะติ) = กิน
- **Vattati** (วัตตะติ) = ใช้ / เป็นไป (ใช้กับกฎหมาย)
- **Vasati** (วะสะติ) = อยู่ / อาศัย
- **Coreti** (โจเรติ) = ขโมย
- **Gaṇhāti** (คัณหาติ) = ยึดถือ / จับ
- **Hanati** (หะนะติ) = ฆ่า / ทำร้าย
- **Dadāti** (ทะทาติ) = ให้
- **Deti** (เทติ) = จ่าย (รูปย่อของ Dadāti)
- **Icchati** (อิจฉะติ) = ต้องการ
- **Kiṇāti** (กิณาติ) = ซื้อ
- **Jānāti** (ชานาติ) = รู้
- **Lakkhati** (ลักขะติ) = รักษา / เฝ้า
- **Nīharati** (นีหะระติ) = นำออก
- **Saṅkamati** (สังกะมะติ) = โอน / ย้าย
- **Otarati** (โอตะระติ) = ลดลง / ข้ามลง
- **Paṭidadati** (ปะฏิทะทะติ) = ทอน (ให้คืน)
- **Sañcarati** (สัญจะระติ) = เดิน

- **Purisa** (ปุริสะ) = ผู้ชาย
- **Ghara** (ฆะระ) = บ้าน
- **Udaka** (อุทะกะ) = น้ำ

- **Puggala** (ปุคคะละ) = บุคคล / คน

- **Daṇḍa** (ทัณฑะ) = โทษ

- **Niyāma** (นิยามะ) = นิยาม
- **Naya** (นะยะ) = ให้ / คำบัญชา / บทกฎหมาย
- **Dhana** (ธะนะ) = ทรัพย์ / เงิน
- **Pitā** (ปิตา) = พ่อ
- **Mātā** (มาตา) = แม่
- **Kicca** (กิจจะ) = หน้าที่ / ธุระ
- **Mātikā** (มาติกา) = มาตรา / แม่บท

- **Sacca** (สัจจะ) = ความจริง
- **Cora** (โจระ) = โจร
- **Rūpiya** (รูปิยะ) = หน่วยเงิน (รูปี)

- **Kahāpaṇa** (กะหาปะณะ) = เหรียญ

- **Paññatta** (ปัญญัตตะ) = บทกำหนด
- **Paṭidada** (ปะฏิทะทะ) = การทอน (การให้คืน)

## 5. หมวดคำเชื่อมและบุพบท (Prepositions & Conjunctions)
- **Nī** (นี) = ใน / ที่ / สู่ / ยัง
- **Cī** (จี) = จาก
- **Sa** (สะ) = ของ
- **Se** (เส) = ด้วย (โดยใช้เครื่องมือ)
- **Saha** (สะหะ) = กับ / ร่วมกับ (ไปกับบุคคล)
- **Yadi** (ยะทิ) = หากว่า
- **Yasmā** (ยัสมา) = เพราะว่า
- **Tasmā** (ตัสมา) = ดังนั้น
- **Aññatra** (อัญญัตตระ) = เว้นแต่
- **Ca** (จะ) = และ
- **Vā** (วา) = หรือ
- **Pana** (ปะนะ) = แต่ / ส่วนว่า

---

## 6. หมวดกาลเวลาและคำช่วย (Time & Particles)
- **Vatta** (วัตตะ) = กำลัง (Present Continuous)
- **Bhavis** (ภะวิส) = จะ (Future)
- **Tvā** (ตวา) = แล้ว (Past/Perfect)
- **Labha** (ละภะ) = ได้ (Auxiliary verb)
- **Etarahi** (เอตะระหิ) = ตอนนี้
- **Pubba** (ปุพพะ) = ก่อน / แล้ว
- **Hiyyo** (หิยโย) = เมื่อวาน
- **Anāgata** (อะนาคะตะ) = อนาคต
- **Sve** (สเว) = พรุ่งนี้
- **Sakka** (สักกะ) = ได้ (วางหลังกิริยา)

- **Aṭṭha** (อัฏฐะ) = 8
- **Nava** (นะวะ) = 9
- **Dasa** (ทะสะ) = 10
- **Sata** (สะตะ) = 100
- **Sahassa** (สะหัสสะ) = 1,000
- **Lakkha** (ลักขะ) = 100,000
- **Niyuta** (นิยุตะ) = 1,000,000
- **Koṭi** (โกฏิ) = 10,000,000
