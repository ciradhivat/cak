# ตำราภาษาผัก ฉบับมาตรฐาน

**เวอร์ชัน 3.12 – สำหรับการสื่อสารและร่างกฎหมาย**

---

## คำนำ

ภาษาผัก (Bhāsā Phak) เป็นภาษาประดิษฐ์ที่พัฒนาขึ้นเพื่อใช้เป็นเครื่องมือในการสื่อสารและร่างเอกสารทางกฎหมายอย่างชัดเจนและแม่นยำ โดยมีรากฐานมาจากภาษาบาลีและโครงสร้างไวยากรณ์ไทยที่เรียบง่าย เหมาะสำหรับผู้ใช้ที่ต้องการความแม่นยำในการสื่อสารทางราชการ

**จุดเด่นของภาษาผัก:**
- โครงสร้างประโยคเรียบง่าย ใกล้เคียงภาษาไทย
- ไม่มีการผันรูปคำกริยาตามกาล ใช้คำช่วยแทน
- รากศัพท์จากภาษาบาลี คนไทยเข้าใจได้ง่าย
- เหมาะสำหรับการร่างกฎหมายและเอกสารทางการ

## ส่วนที่ 1: ระบบพื้นฐาน

### บทที่ 1 – ระบบเสียงและการออกเสียง

#### 1.1 สระ (Vowels)

ภาษาผักใช้ระบบสระ 8 เสียง แบ่งเป็นสระสั้นและสระยาว:

| สระ | การออกเสียง | ตัวอย่าง |
|-----|-------------|----------|
| อะ | a (สั้น) | karati (กะระติ) |
| อา | ā (ยาว) | mātā (มาตา) |
| อิ | i (สั้น) | mitta (มิตตะ) |
| อี | ī (ยาว) | rūpī (รูปี) |
| อุ | u (สั้น) | kusala (กุสะละ) |
| อู | ū (ยาว) | rūpiya (รูปิยะ) |
| เอ | e | deti (เทติ) |
| โอ | o | hoti (โหติ) |

#### 1.2 พยัญชนะ (Consonants)

พยัญชนะแบ่งตามตำแหน่งการออกเสียง:

**กลุ่มกัณฐชะ (Guttural - เกิดที่คอ)**
- ก ข ค ฆ ง → k, kh, g, gh, ṅ

**กลุ่มตาลุชะ (Palatal - เกิดที่เพดาน)**
- จ ฉ ช ฌ ญ → c, ch, j, jh, ñ

**กลุ่มมูรธชะ (Retroflex - เกิดที่ปุ่มเหงือก)**
- ฎ ฏ ฐ ฑ ฒ ณ → ṭ, ṭh, ḍ, ḍh, ṇ

**กลุ่มทันตชะ (Dental - เกิดที่ฟัน)**
- ต ถ ท ธ น → t, th, d, dh, n

**กลุ่มโอฏฐชะ (Labial - เกิดที่ริมฝีปาก)**
- ป ผ พ ภ ม → p, ph, b, bh, m

**กลุ่มอื่นๆ**
- ย ร ล ว ส ห ฬ อํ → y, r, l, v, s, h, ḷ, ṃ

#### 1.3 กฎการออกเสียงพิเศษ

**กฎที่ 1: อิย → อี**
- เขียน: อิย (iy)
- อ่าน: อี (ī)
- ตัวอย่าง: hiyyo (หิยโย) → อ่านว่า "หีโย" (เมื่อวาน)

**กฎที่ 2: อุว → โอ**
- เขียน: อุว (uv)
- อ่าน: โอ (o)
- ตัวอย่าง: yuva (ยุวะ) → อ่านว่า "โย"

**หลักการทั่วไป**
- ออกเสียงทุกพยางค์ที่มีสระและตัวสะกด
- ใช้ระบบเสียงเหมือนภาษาไทย
- สามารถใช้แป้นพิมพ์สันสกฤต (อักษรละติน) หรือ IPA keyboard

### บทที่ 2 – คำกริยาและโครงสร้างประโยค

#### 2.1 โครงสร้างคำกริยา

**หลักสำคัญ:** คำกริยาในภาษาผักไม่มีการผันรูปตามกาล ให้ใช้คำช่วยบอกเวลาแทน

**สูตร:** กริยา = ธาตุ + ปัจจัยท้ายคำ (-ti)

#### 2.2 คำกริยาพื้นฐาน 20 คำ

| ภาษาผัก | การออกเสียง | ความหมาย |
|---------|-------------|----------|
| karati | กะระติ | ย่อมทำ |
| gacchati | คัจฉะติ | ย่อมไป |
| āgacchati | อาคัจฉะติ | ย่อมมา |
| passati | ปัสสะติ | ย่อมเห็น |
| suṇati | สุณะติ | ย่อมฟัง |
| vadati | วะทะติ | ย่อมพูด |
| bhuñjati | ภุญชะติ | ย่อมกิน |
| pivati | ปิวะติ | ย่อมดื่ม |
| vasati | วะสะติ | ย่อมอยู่/อาศัย |
| hoti | โหติ | ย่อมเป็น/อยู่/คือ/มี |
| dadāti | ทะทาติ | ย่อมให้ |
| deti | เทติ | ย่อมให้ (รูปสั้น) |
| harati | หะระติ | ย่อมนำ/พา |
| gaṇhāti | คัณหาติ | ย่อมยึดถือ/จับ |
| icchati | อิจฉะติ | ย่อมต้องการ |
| jānāti | ชานาติ | ย่อมรู้ |
| kiṇāti | กิณาติ | ย่อมซื้อ |
| lakkhati | ลักขะติ | ย่อมรักษา/ปกป้อง |
| hanati | หะนะติ | ย่อมฆ่า/ทำร้าย |
| coreti | โจเรติ | ย่อมขโมย |

#### 2.3 โครงสร้างประโยคพื้นฐาน

**กฎทอง:** โครงสร้างประโยคภาษาผักเหมือนภาษาไทย สามารถแปลคำต่อคำได้

**โครงสร้างมาตรฐาน:**
```
[ประธาน] + [กริยา] + [กรรม] + [ส่วนขยาย]
```

**ตัวอย่าง:**
```
Aha passati sakuṇa
(อะหะ ปัสสะติ สะกุณะ)
ฉัน เห็น นก
```

**กฎการวางตำแหน่ง:**
1. คำนามหน้ากริยา = ประธาน (Subject)
2. คำนามหลังกริยา = กรรม (Object)
3. คำขยายตามหลังคำหลัก

#### 2.4 คำบุพบท (Prepositions)

คำบุพบทใช้แทนวิภัตติปัจจัยในภาษาบาลี วางหน้าคำนามเสมอ

| ภาษาผัก | ความหมาย | ตัวอย่าง |
|---------|----------|----------|
| nī | ที่/ใน/สู่/ยัง | nī ghara (ใน/ที่บ้าน) |
| cī | จาก | cī ghara (จากบ้าน) |
| sa | ของ | sa purisa (ของผู้ชาย) |
| se | ด้วย | se hattha (ด้วยมือ) |
| saha | กับ/ร่วมกับ | saha mitta (กับเพื่อน) |

**หมายเหตุสำคัญ:** บุพบท "nī" สามารถใช้บอกสิ่งที่จะทำแต่ยังไม่ได้ทำ
- ตัวอย่าง: nī na labha karati (ยังไม่ได้ทำ)

### บทที่ 3 – ระบบกาลเวลา

#### 3.1 ความหมายของ "ได้" สองแบบ

ภาษาผักแยกคำว่า "ได้" ออกเป็น 2 ประเภทอย่างชัดเจน:

**3.1.1 Labha (ละภะ) = ได้ (โอกาส)**
- **ความหมาย:** ได้, ได้รับ, มีโอกาสได้
- **ตำแหน่ง:** วาง**หน้า**กริยา
- **รากศัพท์:** มาจากภาษาบาลี แปลว่า "ลาภ"
- **ตัวอย่าง:** Labha karati (ได้ทำ = มีโอกาสทำ)

**3.1.2 Sakka (สักกะ) = ได้ (ความสามารถ)**
- **ความหมาย:** สามารถ, ทำได้, ไหว
- **ตำแหน่ง:** วาง**หลัง**กริยา
- **รากศัพท์:** มาจากภาษาบาลี Sakka (อาจ/สามารถ)
- **ตัวอย่าง:** Karati sakka (ทำได้ = มีความสามารถทำ)

**กฎการจำ:**
- **หน้า-ลาภ** (โอกาส): Labha + กริยา → "มีลาภเลยได้ทำ"
- **หลัง-สัก** (ฝีมือ): กริยา + Sakka → "มีศักยภาพทำได้"

**ตารางเปรียบเทียบ:**

| สถานการณ์ | ภาษาผัก | โครงสร้าง | ความหมาย |
|-----------|---------|-----------|----------|
| ได้ทำ | Labha karati | Labha + V | มีโอกาสได้ทำ |
| ทำได้ | Karati sakka | V + Sakka | มีความสามารถทำ |
| ไม่ได้ทำ | Na labha karati | Na + Labha + V | ไม่มีโอกาสทำ |
| ทำไม่ได้ | Karati na sakka | V + Na + Sakka | ทำไม่เป็น |

**การใช้ร่วมกับ Tvā (แล้ว):**
1. **Labha + กริยา + Tvā** (ได้...แล้ว)
   - ความหมาย: เหตุการณ์เกิดขึ้นและจบลงแล้ว
   - ตัวอย่าง: Aha labha gacchati tvā (ฉันได้ไปแล้ว)

2. **กริยา + Sakka + Tvā** (ทำ...ได้แล้ว)
   - ความหมาย: พัฒนาความสามารถแล้ว
   - ตัวอย่าง: Aha gacchati sakka tvā (ฉันไปได้แล้ว - เช่น ขาหายเจ็บ)

#### 3.2 คำช่วยบอกกาล (Time Particles)

ใช้ระบบคำช่วยวางประกบกริยาแบบภาษาไทย

| ภาษาไทย | ภาษาผัก | รากศัพท์ | ตำแหน่ง |
|---------|---------|----------|---------|
| กำลัง | Vatta | Vattati (ดำเนินไป) | หน้ากริยา |
| จะ | Bhavis | Bhavissati (จักเป็น) | หน้ากริยา |
| แล้ว | Tvā | Tvā (การกระทำสำเร็จ) | หลังกริยา/กรรม |

#### 3.3 ตัวอย่างการผสมคำช่วย

| รูปแบบ | ภาษาผัก | ความหมาย |
|--------|---------|----------|
| อดีตสมบูรณ์ | Bhuñjati tvā | กินแล้ว |
| ปัจจุบันต่อเนื่อง | Vatta bhuñjati | กำลังกิน |
| อนาคต | Bhavis bhuñjati | จะกิน |
| เกือบทำ | Vatta bhavis bhuñjati tvā | กำลังจะกินแล้ว (แต่ไม่ได้กิน) |

#### 3.4 ประโยคสมมติและเหตุการณ์ที่ไม่เกิด

**1. เกือบทำ (Almost did)**

**โครงสร้าง:** Vatta + Bhavis + [กริยา] + Tvā

**ความหมาย:** ตั้งท่าจะทำแต่สุดท้ายไม่ได้ทำ (= Na labha [กริยา])

**ตัวอย่าง:**
```
Vatta bhavis bhuñjati tvā
= กำลังจะกินแล้ว (แต่ไฟไหม้บ้านเลยอดกิน)
= Na labha bhuñjati (ไม่ได้กิน)
```

**2. จะทำแล้ว (Was going to do)**

**โครงสร้าง:** Bhavis + [กริยา] + Tvā

**ตัวอย่าง:**
```
Bhavis gacchati tvā
= จะไปแล้ว (แต่ฝนตกเลยไม่ได้ไป)
```

#### 3.5 คำบอกเวลาเฉพาะ

| ภาษาผัก | การออกเสียง | ความหมาย |
|---------|-------------|----------|
| etarahi | เอตะระหิ | ตอนนี้/กำลัง |
| pubba | ปุพพะ | ก่อน/แล้ว |
| hiyyo | หีโย | เมื่อวาน |
| sve | เสว | พรุ่งนี้ |
| kadā | กะทา | เมื่อไหร่ |
| anāgata | อะนาคะตะ | อนาคต/จะ |

**หมายเหตุ:** ในประโยคคำถามที่มีคำบอกเวลาอดีต การเติม "Tvā" เป็นทางเลือก (ไม่บังคับ)

## ส่วนที่ 2: โครงสร้างประโยคขั้นสูง

### บทที่ 4 – ประโยคซับซ้อนและคำเชื่อม

#### 4.1 คำเชื่อมอนุประโยค (Relative Pronoun)

**Ya (ยะ)** = ผู้/ที่/ซึ่ง/อัน

**หน้าที่:** ใช้ขยายคำนามเพื่อให้รายละเอียดเพิ่มเติม

**ตำแหน่ง:** วางหลังคำนามที่ต้องการขยาย

**โครงสร้าง:**
```
[คำนามหลัก] + ya + [ประโยคขยาย]
```

**ตัวอย่างที่ 1: ขยายประธาน**
```
Puggala ya coreti dhana
(ปุคคะละ ยะ โจเรติ ธะนะ)
= บุคคล ผู้ ขโมย ทรัพย์
```

**ตัวอย่างที่ 2: ขยายกรรม**
```
Dhana ya so gaṇhāti
(ธะนะ ยะ โส คัณหาติ)
= ทรัพย์ ที่ เขา ยึดถือ
```

#### 4.2 คำเชื่อมประโยค (Conjunctions)

| ภาษาผัก | การออกเสียง | ความหมาย | การใช้งาน |
|---------|-------------|----------|-----------|
| ca | จะ | และ | เชื่อมความคล้อยตามกัน |
| vā | วา | หรือ | เชื่อมความให้เลือก |
| pana | ปะนะ | แต่/ส่วน | เชื่อมความขัดแย้ง |

**ตัวอย่าง:**

**1. Ca (และ):**
```
Pitā ca mātā
= พ่อ และ แม่
```

**2. Vā (หรือ):**
```
Tava icchati odana vā maṃsa
= เธอ ต้องการ ข้าว หรือ เนื้อ
```

**3. Pana (แต่):**
```
So icchati dhana pana na karati kicca
= เขา ต้องการ เงิน แต่ ไม่ ทำ หน้าที่
```

#### 4.3 การวางตำแหน่งคำบอกเวลา

เพื่อความง่ายในประโยคซับซ้อน สามารถวางคำบอกเวลาที่**หน้าประธาน/กริยา**หรือ**หลังกรรม** ก็ได้ตามความนิยม

**ตัวอย่าง:**
```
✓ Sve so dadāti dhana
✓ So dadāti dhana sve
✓ So sve dadāti dhana 
```

### บทที่ 5 – ประโยคเงื่อนไขและเหตุผล

#### 5.1 คำเชื่อมเงื่อนไขและเหตุผล

| ภาษาผัก | การออกเสียง | ความหมาย |
|---------|-------------|----------|
| yadi | ยะทิ | หากว่า |
| yasmā | ยัสมา | เพราะว่า |
| tasmā | ตัสมา | ดังนั้น |
| aññatra | อัญญัตตระ | เว้นแต่ |
| kasmā | กัสมา | ทำไม |

#### 5.2 โครงสร้างประโยคเงื่อนไข

**รูปแบบพื้นฐาน:**
```
Yadi [เงื่อนไข], Tasmā [ผลลัพธ์]
```

**ตัวอย่าง:**
```
Yadi puggala na karati kusala, tasmā naya hoti daṇḍa

แปล: หากบุคคลไม่ทำดี ดังนั้นให้มีโทษ
```

#### 5.3 ประโยคเงื่อนไขซับซ้อน

ใช้ร่วมกับ "ya" เพื่อระบุตัวบุคคลหรือสิ่งของให้ชัดเจน

**โครงสร้าง:**
```
Yadi [ประธาน + ya + ส่วนขยาย] [กริยา] [กรรม], Tasmā [ผลลัพธ์]
```

**ตัวอย่าง:**
```
Yadi puggala ya hanati sakuṇa, tasmā naya hoti daṇḍa

วิเคราะห์:
- Yadi = หาก
- Puggala = บุคคล
- Ya = ผู้ (เริ่มส่วนขยาย)
- Hanati = ฆ่า
- Sakuṇa = นก (จบส่วนขยาย)
- Tasmā = ดังนั้น
- Naya = ให้ (บทบัญญัติ)
- Hoti = มี
- Daṇḍa = โทษ

แปล: "หากบุคคลผู้ฆ่านก ดังนั้นให้มีโทษ"
```

#### 5.4 โครงสร้างเหตุและผล

**ตัวอย่าง:**
```
คำถาม: Kasmā so coreti?
        (ทำไมเขาขโมย?)

คำตอบ: Yasmā so icchati dhana
        (เพราะว่าเขาต้องการเงิน)
```

### บทที่ 6 – ประโยคคำถาม

#### 6.1 คำถามพื้นฐาน (Question Words)

**หลักจำง่าย:** คำถามส่วนใหญ่ขึ้นต้นด้วยตัว **K**

| ภาษาไทย | ภาษาผัก | การออกเสียง | หมายเหตุ |
|---------|---------|-------------|---------|
| ใคร | Ko | โก | วางตำแหน่งเหมือนภาษาไทย |
| อะไร | Ki | กิ | วางตำแหน่งเหมือนภาษาไทย |
| ที่ไหน | Kuttha | กุตถะ | วางตำแหน่งเหมือนภาษาไทย |
| เมื่อไหร่ | Kadā | กะทา | วางตำแหน่งเหมือนภาษาไทย |
| ทำไม | Kasmā | กัสมา | วางตำแหน่งเหมือนภาษาไทย |
| อย่างไร | Katha | กะถะ | วางตำแหน่งเหมือนภาษาไทย |
| ไหม/หรือ | Nu | นุ | วางตำแหน่งเหมือนภาษาไทย |

#### 6.2 โครงสร้างและตัวอย่าง

**1. Ko (ใคร)**

ถามผู้กระทำ (วางหน้ากริยา):
```
Ko lakkhati ghara?
= ใคร รักษา บ้าน?

→ Pitā lakkhati ghara
   = พ่อ รักษา บ้าน
```

ถามผู้ถูกกระทำ (วางหลังกริยา):
```
Tava passati ko?
= เธอ เห็น ใคร?

→ Aha passati cora
   = ฉัน เห็น โจร
```

**2. Ki (อะไร)**
```
So kiṇāti ki?
= เขา ซื้อ อะไร?

→ So kiṇāti odana
   = เขา ซื้อ ข้าว
```

**3. Kuttha (ที่ไหน)**
```
Cora vasati kuttha?
= โจร พักอยู่ ที่ไหน?
```

ใช้กับบุพบท:
```
Tava harati dhana bahi cī kuttha?
= เธอ นำ เงิน ออก จาก ที่ไหน?
```

**4. Kadā (เมื่อไหร่)**
```
So āgacchati kadā?
= เขา มา เมื่อไหร่?

→ So āgacchati sve
   = เขา มา พรุ่งนี้
```

**5. Kasmā (ทำไม)**
```
Kasmā so coreti?
= ทำไม เขา ขโมย?

→ Yasmā so icchati dhana
   = เพราะว่า เขา ต้องการ เงิน
```

**6. Katha (อย่างไร)**
```
Tava jānāti sacca katha?
= เธอ รู้ ความจริง อย่างไร?

→ Aha suṇati
   = ฉัน ฟัง
```

**7. Nu (ไหม/หรือไม่)**
```
Tava hoti cora nu?
= เธอ คือ โจร ใช่ไหม?

→ Āma (ใช่) / Na (ไม่)
```

#### 6.3 คำตอบพื้นฐาน

| ภาษาผัก | การออกเสียง | ความหมาย |
|---------|-------------|----------|
| Āma | อามะ | ใช่/ครับ/ค่ะ |
| Na | นะ | ไม่ |

### บทที่ 7 – ระบบตัวเลขและลำดับ

#### 7.1 เลขจำนวนนับ (Cardinal Numbers)

**หลักหน่วย (0-9)**

| เลข | ภาษาผัก | การออกเสียง |
|-----|---------|-------------|
| 0 | Suñña | สุญญะ |

| 8 | Aṭṭha | อัฏฐะ |
| 9 | Nava | นะวะ |

**หลักสิบ (10-99)**

**สูตร:** [เลขหน้า] + Dasa + [เลขหลัง]

| เลข | ภาษาผัก | คำอ่าน |
|-----|---------|--------|
| 10 | Dasa | ทะสะ |
| 11 | Dasa eka | ทะสะ-เอกะ |

| 20 | Dvi dasa | ทะวิ-ทะสะ |
| 21 | Dvi dasa eka | ทะวิ-ทะสะ-เอกะ |
| 45 | Catu dasa pañca | จะตุ-ทะสะ-ปัญจะ |
| 99 | Nava dasa nava | นะวะ-ทะสะ-นะวะ |

**หลักใหญ่**

| จำนวน | ภาษาผัก | การออกเสียง |
|-------|---------|-------------|
| 100 | Sata | สะตะ |
| 1,000 | Sahassa | สะหัสสะ |
| 10,000 | Dasa sahassa | ทะสะ-สะหัสสะ |
| 100,000 | Lakkha | ลักขะ |
| 1,000,000 | Niyuta | นิยุตะ |
| 10,000,000 | Koṭi | โกฏิ |

**ตัวอย่างการอ่านจำนวนเต็ม:**
- **356** = Ti sata pañca dasa cha (สาม-ร้อย-ห้า-สิบ-หก)
- **2,500** = Dvi sahassa pañca sata (สอง-พัน-ห้า-ร้อย)
- **1,000,000** = Niyuta (หนึ่ง-ล้าน)

| ที่ 10 | Kama dasa | กะมะ ทะสะ |
| ที่ 11 | Kama dasa eka | กะมะ ทะสะ-เอกะ |
| ที่ 100 | Kama sata | กะมะ สะตะ |

**การใช้งาน:**
- **มาตราที่ 1** = Mātikā kama eka
- **คนที่ 2** = Puggala kama dvi

#### 7.3 ทศนิยมและเศษส่วน

**1. ทศนิยม**

**คำสำคัญ:** Bindu (พินทุ) = จุด

**ตัวอย่าง:**

- **3.14** = Ti bindu eka catu
- **99.9** = Nava dasa nava bindu nava

|---------|---------|-------------|

| หนึ่งในสี่ (1/4) | Pāda | ปาทะ |
| ส่วน/ภาค | Bhāga | ภาคะ |

**ตัวอย่างเศษส่วน:**
- **1/3** = Eka bhāga ti (หนึ่ง ส่วน สาม)
- **2/5** = Dvi bhāga pañca (สอง ส่วน ห้า)
- **3/4** = Ti bhāga catu (สาม ส่วน สี่)

### บทที่ 8 – ระบบเงินตรา

#### 8.1 หน่วยเงินตรา (Currency Unit)

**หน่วยหลัก:** Rūpiya (รู-ปิ-ยะ) หรือ Rūpī (รู-ปี)
- **สัญลักษณ์:** Rp. หรือ ₹
- **รากศัพท์:** Rūpiya (เงินตรา/รูปพรรณ)

**หน่วยย่อย:** Aṇu (อะ-ณุ)
- **อัตราแลกเปลี่ยน:** 100 Aṇu = 1 Rūpiya
- **รากศัพท์:** Aṇu (อนุภาค/สิ่งเล็กๆ)
- **ระบบ:** ทศนิยม (เหมือนเซนต์หรือสตางค์)

#### 8.2 ประเภทเงินตามรูปแบบ

| ประเภท | ภาษาผัก | การออกเสียง | ความหมาย |
|--------|---------|-------------|----------|
| เหรียญ | Kahāpaṇa | กะหาปะณะ | เหรียญกษาปณ์ |
| ธนบัตร | Dhanapatta | ธะนะปัตตะ | ใบทรัพย์ |
| เงินสด | Dhanahattha | ธะนะหัตถะ | ทรัพย์ในมือ |
| เงินดิจิทัล | Dhanaākāsa | ธะนะอากาสะ | ทรัพย์ในอากาศ |

#### 8.3 มูลค่าธนบัตรและเหรียญ

ใช้ระบบ 1-2-5-10 (มาตรฐานสากล)

**เหรียญ (Kahāpaṇa):**
- 1 Rp. = Eka rūpiya
- 5 Rp. = Pañca rūpiya
- 10 Rp. = Dasa rūpiya

**ธนบัตร (Dhanapatta):**
- 50 Rp. = Pañca dasa rūpiya
- 100 Rp. = Sata rūpiya
- 500 Rp. = Pañca sata rūpiya
- 1,000 Rp. = Sahassa rūpiya

#### 8.4 การอ่านราคา

**โครงสร้าง:** [ตัวเลข] + [หน่วยหลัก] + [ตัวเลข] + [หน่วยย่อย]

**ตัวอย่าง:**

**1. ราคาเต็ม:**
```
150 Rp. = Sata pañca dasa rūpiya
(ร้อย-ห้า-สิบ-รูปิยะ)
```

**2. ราคามีทศนิยม:**
```
20.50 Rp. = Dvi dasa rūpiya pañca dasa aṇu
(สอง-สิบ-รูปิยะ-ห้า-สิบ-อะณุ)
```

**3. ภาษาพูดแบบย่อ:**
```
20.50 Rp. = Dvi dasa bindu pañca dasa
(20 จุด 50)
[ตัด aṇu ทิ้งได้หากเป็นที่เข้าใจกัน]
```

#### 8.5 คำศัพท์ธุรกรรมการเงิน

| ภาษาไทย | ภาษาผัก | การออกเสียง | รากศัพท์ |
|---------|---------|-------------|----------|
| ราคา | Mūla | มูละ | มูลค่า/รากฐาน |
| ถูก | Appa | อัปปะ | น้อย |
| แพง | Mahaggha | มะหัคฆะ | ค่า มาก |
| ลดราคา | Otarati | โอตะระติ | ข้ามลง/ลดลง |
| จ่าย | Deti | เทติ | ให้ (รูปสั้น) |
| ทอน | Paṭidada | ปะฏิทะทะ | ให้คืน |
| โอน | Saṅkamati | สังกะมะติ | ก้าวไป/ย้าย |
| บัญชี | Lekha | เลขะ | การจด/เขียน |
| ทรัพย์/เงิน | Dhana | ธะนะ | ทรัพย์สมบัติ |
| เท่าไหร่ | Ki | กิ | (ใช้ในคำถาม) |
| ถ้วน/บริบูรณ์ | Paripuṇṇa | ปะริปุณณะ | สมบูรณ์ |

**ตัวอย่างประโยค:**
- **Mūla ki?** = ราคาเท่าไหร่?
- **Mūla appa** = ราคาถูก
- **Naya otarati mūla** = ให้ลดราคา
- **Aha deti dhanahattha** = ฉันจ่ายเงินสด
- **Tava paṭidada rūpiya** = เธอทอนเงิน

#### 8.6 บทสนทนาซื้อขาย

**สถานการณ์ที่ 1: ถามราคา**
```
ลูกค้า: Mūla ki?
       (ราคา เท่าไหร่?)

ผู้ขาย: Sata rūpiya paripuṇṇa
       (ร้อย รูปิยะ ถ้วน)
```

**สถานการณ์ที่ 2: จ่ายเงิน**
```
ลูกค้า: Aya hoti dhanahattha
       (นี่ คือ เงินสด)

ผู้ขาย: Aha paṭidada dvi dasa rūpiya
       (ฉัน ทอน สอง-สิบ รูปิยะ)
```

**สถานการณ์ที่ 3: ต่อรองราคา**
```
ลูกค้า: Mūla mahaggha. Otarati mūla labha nu?
       (ราคา แพง. ลด ราคา ได้ ไหม?)

ผู้ขาย: Āma. Labha pañca dasa rūpiya paripuṇṇa
       (ใช่. ได้ ห้า-สิบ รูปิยะ ถ้วน)
```

### บทที่ 10 – ตัวอย่างบทกฎหมาย

#### 10.1 โครงสร้างมาตราพื้นฐาน

**มาตราที่ 1: บทบัญญัติทั่วไป**
```
Mātikā kama eka

Niyāma aya vattati sabba

แปล:
มาตรา ที่ หนึ่ง
บทบัญญัติ นี้ ใช้ ทั่วไป
```

**มาตราที่ 2: คำสั่งเฉพาะ**
```
Mātikā kama dvi

Naya puggala vasati nī ghara

แปล:
มาตรา ที่ สอง
ให้ บุคคล อยู่ ใน บ้าน
```

**มาตราที่ 3: บทลงโทษ**
```
Mātikā kama ti

Yadi puggala na karati kusala,
Tasmā naya hoti daṇḍa

แปล:
มาตรา ที่ สาม
หาก บุคคล ไม่ ทำ ดี,
ดังนั้น ให้ มี โทษ
```

#### 10.2 กฎหมายที่มีอนุประโยค

**ตัวอย่าง: กฎหมายคุ้มครองสัตว์**
```
Mātikā kama catu

Yadi puggala ya hanati sakuṇa,
Tasmā naya hoti daṇḍa sata rūpiya

วิเคราะห์:
- Mātikā kama catu = มาตราที่สี่
- Yadi = หาก
- Puggala ya hanati sakuṇa = บุคคลผู้ฆ่านก
- Tasmā = ดังนั้น
- Naya hoti daṇḍa = ให้มีโทษ
- Sata rūpiya = ร้อยรูปิยะ

แปล:
"หากบุคคลผู้ฆ่านก ดังนั้นให้มีโทษหนึ่งร้อยรูปิยะ"
```

#### 10.3 กฎหมายที่มีข้อยกเว้น

**ตัวอย่าง: กฎหมายทรัพย์สิน**
```
Mātikā kama pañca

Naya puggala na gaṇhāti dhana sa puggala añña,
Aññatra so dadāti āma

วิเคราะห์:
- Naya = ให้
- Puggala na gaṇhāti = บุคคลไม่ยึดถือ
- Dhana sa puggala añña = ทรัพย์ของบุคคลอื่น
- Aññatra = เว้นแต่
- So dadāti āma = เขาให้ความยินยอม

แปล:
"ให้บุคคลไม่ยึดถือทรัพย์ของบุคคลอื่น เว้นแต่เขาให้ความยินยอม"
```

#### 10.4 กฎหมายที่มีหลายเงื่อนไข

**ตัวอย่าง: เงื่อนไขการโอนเงิน**
```
Mātikā kama cha

Yadi puggala ya sambhava saṅkamati dhana,
Ca puggala hoti lekha dhana,
Tasmā naya labha karati

วิเคราะห์:
- Yadi puggala ya sambhava saṅkamati dhana = หากบุคคลผู้อาจโอนเงิน
- Ca puggala hoti lekha dhana = และบุคคลมีบัญชีเงิน
- Tasmā naya labha karati = ดังนั้นให้ได้ทำ

แปล:
"หากบุคคลผู้อาจโอนเงิน และบุคคลมีบัญชีเงิน ดังนั้นให้ได้ทำ"
```

#### 11.1 บทสนทนาทั่วไป

**สถานการณ์: ทักทาย**
```
A: Ko tava?
   (ใคร เธอ? = คุณเป็นใคร?)

B: Aha hoti Somjāȳ
   (ฉัน คือ สมชาย)

A: Tava vasati kuttha?
   (เธอ อยู่ ที่ไหน?)

B: Aha vasati nī kruṅdeb
   (ฉัน อยู่ ใน กรุงเทพ)
```

#### 11.2 บทสนทนาการสอบปากคำ

**สถานการณ์: ตำรวจสอบสวน**
```
ตำรวจ: Hiyyo tava gacchati kuttha?
       (เมื่อวาน เธอ ไป ที่ไหน?)

ผู้ต้องสงสัย: Aha gacchati nī āpaṇa
            (ฉัน ไป ที่ ตลาด)

ตำรวจ: Tava gacchati saha ko?
       (เธอ ไป กับ ใคร?)

ผู้ต้องสงสัย: Gacchati saha mitta
            (ไป กับ เพื่อน)

ตำรวจ: Kasmā tava hanati so?
       (ทำไม เธอ ฆ่า เขา?)

ผู้ต้องสงสัย: Na! Aha na labha hanati!
            (ไม่! ฉัน ไม่ ได้ ฆ่า!)
```

#### 11.3 บทสนทนาการซื้อขาย

**สถานการณ์: ในร้านอาหาร**
```
ลูกค้า: Aya hoti ki?
       (นี่ คือ อะไร?)

พนักงาน: Aya hoti odana ca maṃsa
        (นี่ คือ ข้าว และ เนื้อ)

ลูกค้า: Mūla ki?
       (ราคา เท่าไหร่?)

พนักงาน: Labha cha dasa rūpiya
        (ได้ หก-สิบ รูปิยะ = 60 รูปิยะ)

ลูกค้า: Mūla mahaggha. Otarati mūla labha nu?
       (ราคา แพง. ลด ราคา ได้ ไหม?)

พนักงาน: Āma. Labha pañca dasa rūpiya paripuṇṇa
        (ใช่. ได้ ห้า-สิบ รูปิยะ ถ้วน = 50 รูปิยะถ้วน)
```

#### 11.4 บทสนทนาเกี่ยวกับเวลา

**สถานการณ์: นัดหมาย**
```
A: Tava āgacchati kadā?
   (เธอ มา เมื่อไหร่?)

B: Aha bhavis āgacchati sve
   (ฉัน จะ มา พรุ่งนี้)

A: Sve tava āgacchati katha?
   (พรุ่งนี้ เธอ มา อย่างไร?)

B: Aha gacchati se ratha
   (ฉัน ไป ด้วย รถ)

A: Āma. Aha bhavis passati tava sve
   (ใช่. ฉัน จะ เห็น เธอ พรุ่งนี้)
   [หมายถึง: ฉันจะเจอเธอพรุ่งนี้]
```

#### 11.5 บทสนทนาซับซ้อน

**สถานการณ์: อธิบายเหตุการณ์**
```
A: Kasmā so na āgacchati hiyyo?
   (ทำไม เขา ไม่ มา เมื่อวาน?)

B: Yasmā so vatta bhavis āgacchati tvā,
   Pana ratha na sañcarati
   (เพราะว่า เขา กำลัง จะ มา แล้ว,
   แต่ รถ ไม่ เดิน)

A: Tasmā so na labha āgacchati nu?
   (ดังนั้น เขา ไม่ ได้ มา ใช่ไหม?)

B: Āma. So na labha āgacchati
   (ใช่. เขา ไม่ ได้ มา)
```

## แบบฝึกหัดท้ายตำรา

### แบบฝึกหัดที่ 1: แปลประโยคพื้นฐาน

แปลประโยคต่อไปนี้เป็นภาษาผัก:

1. ฉันเห็นนก
2. เขากินข้าวในบ้าน
3. พ่อและแม่อยู่ที่บ้าน
4. เธอซื้ออะไร?
5. โจรขโมยเงิน

**เฉลย:**
1. Aha passati sakuṇa
2. So bhuñjati odana nī ghara
3. Pitā ca mātā vasati nī ghara
4. Tava kiṇāti ki?
5. Cora coreti dhana

---

### แบบฝึกหัดที่ 2: กาลเวลา

แปลประโยคต่อไปนี้เป็นภาษาผัก:

1. ฉันกำลังกิน
2. เขาจะมาพรุ่งนี้
3. เธอกินแล้ว
4. กำลังจะไปแล้ว (แต่ไม่ได้ไป)
5. เมื่อวานเขาไปตลาด

**เฉลย:**
1. Aha vatta bhuñjati
2. So bhavis āgacchati sve
3. Tava bhuñjati tvā
4. Vatta bhavis gacchati tvā (= Na labha gacchati)
5. Hiyyo so gacchati nī āpaṇa

---

### แบบฝึกหัดที่ 3: ประโยคซับซ้อน

แปลประโยคต่อไปนี้เป็นภาษาผัก:

1. คนผู้พูดเท็จ
2. พ่อและแม่กินข้าวในบ้าน
3. หากโจรขโมยของ ดังนั้นโจรมีโทษ
4. เขาต้องการเงินแต่ไม่ทำหน้าที่
5. บุคคลที่เธอเห็นเมื่อวาน

**เฉลย:**
1. Puggala ya vadati musā (musā = เท็จ - คำเพิ่มเติม)
2. Pitā ca mātā bhuñjati odana nī ghara
3. Yadi cora coreti dhana, tasmā cora hoti daṇḍa
4. So icchati dhana pana na karati kicca
5. Puggala ya tava passati hiyyo

---

### แบบฝึกหัดที่ 4: คำถาม

สร้างคำถามจากประโยคต่อไปนี้:

1. เขาอยู่ที่บ้าน → (ถามว่า "ที่ไหน")
2. ฉันมาพรุ่งนี้ → (ถามว่า "เมื่อไหร่")
3. เธอเห็นโจร → (ถามว่า "ใคร")
4. เขาขโมยเพราะต้องการเงิน → (ถามว่า "ทำไม")
5. เขาคือโจร → (ถามแบบ Yes/No)

**เฉลย:**
1. So vasati kuttha?
2. Tava āgacchati kadā?
3. Tava passati ko?
4. Kasmā so coreti?
5. So hoti cora nu?

### แบบฝึกหัดที่ 6: เขียนบทกฎหมาย

เขียนมาตรากฎหมายต่อไปนี้เป็นภาษาผัก:

**มาตราที่ 5:**
"หากบุคคลผู้ขโมยทรัพย์ของผู้อื่น ดังนั้นให้มีโทษหนึ่งพันรูปิยะ เว้นแต่เขาได้รับความยินยอม"

**เฉลย:**
```
Mātikā Kama Pañca

Yadi puggala ya coreti dhana sa puggala añña,
Tasmā naya hoti daṇḍa Sahassa Rūpiya,
Aññatra so labha āma
```

**คำแปลทีละส่วน:**
- Mātikā Kama Pañca = มาตราที่ ๕
- Yadi puggala ya coreti dhana sa puggala = หากบุคคลผู้ขโมยทรัพย์ของบุคคล
- Tasmā naya hoti daṇḍa Sahassa Rūpiya = ดังนั้นให้มีโทษหนึ่งพันรูปิยะ
- Aññatra so labha āma = เว้นแต่เขาได้รับความยินยอม

### ก. ตารางผันคำช่วยกาล

| กาล | โครงสร้าง | ตัวอย่าง | ความหมาย |
|-----|-----------|----------|----------|
| ปัจจุบันธรรมดา | [กริยา] | Bhuñjati | กิน |
| ปัจจุบันต่อเนื่อง | Vatta + [กริยา] | Vatta bhuñjati | กำลังกิน |
| อนาคต | Bhavis + [กริยา] | Bhavis bhuñjati | จะกิน |
| อดีตสมบูรณ์ | [กริยา] + Tvā | Bhuñjati tvā | กินแล้ว |
| อดีตกำลังทำ | Vatta + [กริยา] + (เวลา) | Vatta bhuñjati hiyyo | กำลังกินเมื่อวาน |


**2. ประโยคคำถาม (Wh-Question):**
```
[คำถาม] + [ประธาน] + [กริยา] + [กรรม]?
```

**3. ประโยคคำถาม (Yes/No):**
```
[ประโยคบอกเล่า] + Nu?
```

```
[คำนามหลัก] + Ya + [ส่วนขยาย] + [ส่วนที่เหลือของประโยค]
```

### ค. เทคนิคการจำคำศัพท์

**1. จำตามหมวดหมู่:**
- คำที่ขึ้นต้นด้วย K มักเป็นคำถาม (Ko, Ki, Kuttha, Kadā, Kasmā, Katha)
- คำลงท้าย -ti มักเป็นกริยา (Karati, Gacchati, Passati)
- คำที่มี "dhana" เกี่ยวกับเงิน/ทรัพย์

**2. จำตามรากศัพท์:**
- Vatta → เกี่ยวกับการเป็นไป/ดำเนิน
- Bhavis → เกี่ยวกับอนาคต
- Tvā → เกี่ยวกับความสำเร็จ

**3. จำตามคู่ตรงข้าม:**
- Āma (ใช่) ↔ Na (ไม่)
- Appa (น้อย/ถูก) ↔ Mahaggha (มาก/แพง)
- Kusala (ดี) ↔ Pāpa (ชั่ว - คำเพิ่มเติม)

## สรุป: หลักการใช้ภาษาผัก

### 1. หลักทองสำหรับผู้เรียน

✓ **ใช้โครงสร้างแบบภาษาไทย** - แปลคำต่อคำได้
✓ **กริยาไม่ผันตามกาล** - ใช้คำช่วย Vatta, Bhavis, Tvā
✓ **คำถามขึ้นต้น K** - ง่ายต่อการจำ โก กิ กุ กะ กะ กะ นุ
✓ **คำนามหน้ากริยา = ประธาน** - คำนามหลังกริยา = กรรม
✓ **บุพบทวางหน้าคำนาม** - ไม่มีวิภัตติปัจจัย

### 2. ข้อควรระวัง

⚠️ **การออกเสียงพิเศษ:**
- อิย → อี (hiyyo อ่านว่า "หีโย")
- อุว → โอ (yuva อ่านว่า "โย")

⚠️ **ตรรกะของกาล:**
- "Vatta Bhavis [กริยา] Tvā" = ไม่ได้ทำ (เกือบทำ)
- อย่าลืมว่าเป็นเหตุการณ์ที่ไม่เกิดขึ้น

 * หน้า-ลาภ (โอกาส): ใช้ Labha วางหน้ากริยา (Labha Karati = ได้ทำ) จำง่ายๆ ว่า "มีลาภเลยได้ทำ"
 * หลัง-สัก (ฝีมือ): ใช้ Sakka วางหลังกริยา (Karati Sakka = ทำได้) จำง่ายๆ ว่า "มีศักยภาพทำได้"
 * รูปอดีต: เติม Tvā ปิดท้ายตามปกติ (Labha...Tvā = ได้ทำแล้ว / ...Sakka Tvā = ทำได้แล้ว/ทำเป็นแล้ว)

### 3. เส้นทางการเรียนรู้แนะนำ

**ระดับเริ่มต้น:**
1. ท่องคำกริยาพื้นฐาน 10 คำ
2. เรียนคำสรรพนาม (Aha, Tava, So)
3. ฝึกประโยคง่ายๆ แบบ S-V-O

**ระดับกลาง:**
4. เรียนคำช่วยกาล (Vatta, Bhavis, Tvā)
5. เรียนคำถามทั้งหมด (Ko, Ki, Kuttha...)
6. ฝึกประโยคเงื่อนไข (Yadi...Tasmā...)

**ระดับสูง:**
7. เรียนอนุประโยค (Ya)
8. เรียนคำเชื่อม (Ca, Vā, Pana)
9. ฝึกเขียนบทกฎหมายจริง

### 4. แหล่งข้อมูลเพิ่มเติม

- **พจนานุกรมบาลี-ไทย** - สำหรับค้นรากศัพท์
- **ตำราไวยากรณ์บาลี** - เพื่อทำความเข้าใจโครงสร้างต้นแบบ
- **กฎหมายไทย** - สำหรับศึกษารูปแบบการเขียนบทกฎหมาย
